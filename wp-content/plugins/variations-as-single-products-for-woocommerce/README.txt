== Description ==

This plugin makes your product variations available as single products in shop, category & search result pages.

== Installation ==


1. Upload `variations-as-single-products-for-woocommerce` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
